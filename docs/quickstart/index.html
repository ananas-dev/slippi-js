Quickstart

This page tells you all you need to get started with slippi-js.

note

Make sure Node.js is installed on your system.

Before starting#

  1. Create a fresh directory on your disk
  2. Install @slippi/slippi-js, if you don't know how to do it refer to this section.
  3. Inside this new directory, create a file called script.js.
  4. Take the slippi file you want to work with and move it in the newly created directory.
  5. Rename the slippi file to test.slp.

Load the slp file#

Begin by importing SlippiGame from the @slippi/slippi-js module:

script.js
const { SlippiGame } = require("@slippi/slippi-js");

Lets try to load test.slp:

script.js
const game = new SlippiGame("test.slp");

Now we have a SlippiGame object called game. We can get all the data we need from it.

caution

The api documentation is written for typescript.

Get the game settings#

script.js
const settings = game.getSettings();

We just got the game settings of our slippi file, it contains a lot of useful data about the game before it started like the stage id, the players and the game mode.

Lets print it to the console to see what it contains:

script.js
console.log(settings);

You can then run your program by using node script.js, the output should look something like this:
