Class: SlpStream

SlpStream is a writable stream of Slippi data. It passes the data being written in and emits an event based on what kind of Slippi messages were processed.

SlpStream emits two events: "slp-raw" and "slp-command". The "slp-raw" event emits the raw buffer bytes whenever it processes each command. You can manually parse this or write it to a file. The "slp-command" event returns the parsed payload which you can access the attributes.
